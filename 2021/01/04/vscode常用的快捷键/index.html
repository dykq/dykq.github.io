vscode常用的快捷键
發表於 2021-01-04| 更新於 2021-01-04
閱讀量:

vscode常用的快捷键

Ctrl + Shift + N　　打开新的编辑器窗口

Ctrl + Shift + W　　关闭编辑器窗口

Ctrl + Shift + P　　展开全局命令面板

Ctrl + P　　快速打开最近打开的文件

Ctrl + X　　剪切

Ctrl + C　　复制

Ctrl + H　　打开搜索历史记录

F2　　重命名

Alt + up / down　　移动上下行

Shift + Alt + up / down　　在当前行上复制上下行

Ctrl + Enter　　在当前行下插入新的一行

Ctrl + Shift + Enter　　在当前行上插入新的一行

Ctrl + ] / [　　行缩进

Home　　光标跳转到当前行头

End　　光标跳转到当前行尾

Ctr + Home　　跳转到当前页头

Ctrl + End　　跳转到当前页尾

Ctrl + up / down　　行视图上下偏移

Ctrl + Shift + [　　折叠区域代码

Ctrl + Shift + ]　　展开区域代码

Ctrl + k + 0 　　代码收起

Ctrl + k + j　　代码展开

Ctrl + /　　添加单行注释

Shift + Alt + A　　添加多行注释

Shift + Alt + F　　格式化代码

显示快捷键

F11　　切换全屏模式

Ctrl + = / -　　屏幕放大或缩小

Ctrl + B　　侧边栏隐藏

Ctrl + Shift + X　　打开插件市场面板

Ctrl + Shift + V　　预览编译后的Markdown文件

Alt + L + O　　打开浏览器

Alt + Tab　　切换窗口

Windows + Tab　　切换屏幕

Window + D　　显示桌面

Window + M　　所有程序最小化

查找替换快捷键

Ctrl + F　　查找

Ctrl + H　　查找替换

Ctrl + Shift + F　　在整个文件夹中查找
